## Installation

add to your project , and let your IDE to pick it up automatically

```
npm install --save-dev https://github.com/SmartFoxIO/wix-sdk-typescript
``` 
OR
```
yarn add --dev https://github.com/SmartFoxIO/wix-sdk-typescript
```
